矩阵的QR分解

什么矩阵能进行$QR$分解
A列向量线性无关
不是方阵的情况

$A=QR$
$Q$是标准正交矩阵，$R$是上三角矩阵

A的列向量线性无关，每个列向量可以表示成下面的线性组合方式。
即每个列向量可以表示成各个单位正交向量上的投影量的线性组合方式
$$A=
\left[
\begin{matrix}
u1&u2&u3
\end{matrix}
\right]
$$

$QR$分解
$u1=(u1^Tq1)q1+(u1^Tq2)q2+(u1^Tq3)q3$
$u2=(u2^Tq1)q1+(u2^Tq2)q2+(u2^Tq3)q3$
$u3=(u3^Tq1)q1+(u3^Tq2)q2+(u3^Tq3)q3$

其中$u1$,$u2$,$u3$为矩阵$A$的列向量，对这些列向量进行单位化。

github+hexo搭载博客

发表于 2020-06-26 更新于 2020-06-28

首先要有：

  • 注册github账户
  • 安装git
  • 安装node.js
  • 安装hexo

做完上面的工作以后，开始博客的编写和部署

在任意一个盘新建文件夹blog

  1. hexo init
  2. hexo install
  3. npm install hexo-deployer-git –save

在用markdown编写的时候，在本地显示公式没有问题，但是部署到github上面以后公式显示有问题。

公式问题

hexo+github+markdown公式不能正常显示的问题。
在本地预览公式能正常显示，但是把博客部署到gtihub上公式不能正常显示。

尝试了几次，一直没有搞定，只能先在本地写了。
昨天晚上没事又捣鼓了一下，搞到凌晨两点多。发现网上有人说在next主题下能显示公式。
于是换成next主题。

下面是步骤：
当然需要先搭载博客，具体参考网上，已经很多了。

  1. 换成next主题
    这篇文章挺好设置next主题
  2. 把next主题里面的themes/next-theme/_config.yml
    里面的改成:

mathjax:
enable: true
per_page: false

相对路径
