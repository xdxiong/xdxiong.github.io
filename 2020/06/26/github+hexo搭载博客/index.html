github+hexo搭载博客

发表于 2020-06-26 更新于 2020-06-30

首先要有：

  • 注册github账户
  • 安装git
  • 安装node.js
  • 安装hexo

做完上面的工作以后，开始博客的编写和部署

在任意一个盘新建文件夹blog

  1. hexo init
  2. hexo install
  3. npm install hexo-deployer-git –save

在用markdown编写的时候，在本地显示公式没有问题，但是部署到github上面以后公式显示有问题。

公式问题

hexo+github+markdown公式不能正常显示的问题。
在本地预览公式能正常显示，但是把博客部署到gtihub上公式不能正常显示。

尝试了几次，一直没有搞定，只能先在本地写了。
昨天晚上没事又捣鼓了一下，搞到凌晨两点多。发现网上有人说在next主题下能显示公式。
于是换成next主题。

下面是步骤：
当然需要先搭载博客，具体参考网上，已经很多了。

  1. 换成next主题
    这篇文章挺好设置next主题
  2. 把next主题里面的themes/next-theme/_config.yml
    里面的改成:

mathjax:
enable: true
per_page: false

注意：在写矩阵的 时候，如果出现了多行的元素挤在了一行的情况。那么应该是每一行的后面是四个\

图片显示问题

  1. 在source路径下建立一个images文件夹
  2. 把图片放到文件夹中

注意:在标题后面### 应该有个空格
注意:用命令 新建md文件的时候有标题，尽量用这个，hexo new 文件名

路径前面放斜杠和不放斜杠

图片和公式显示问题

在首页公式和图片显示都没有问题，但是点开每个文章，就开始显示不正常了。
