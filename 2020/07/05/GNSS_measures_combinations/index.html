GNSS组合测量模型

对应频率$f_1$和$f_2$的测量值$R_{pi}$和$\Phi_{Li}$ $i=1,2$。

无电离层组合(Ionosphere-free combination)

消除了一阶电离层影响

$$\Phi_{LC}=\frac{f^2_1\Phi_{L1}-f^2_2\Phi_{L2}}{f^2_1-f^2_2}$$

$$R_{LC}=\frac{f^2_1\Phi_{L1}-f^2_2\Phi_{L2}}{f^2_1-f^2_2}$$

卫星钟定义根据$R_{PC}$组合

无距离 组合(Geometry-free combination)

消除了测量的几何部分，除测量噪声和多路径外，留下与频率相关的影响(电离层折射，硬件延迟，相位缠绕)。他能估计电离层电子含量，监测载波相位中的周跳，也能估计天线旋转。
注意$R_{pI}$和$\Phi_{LI}$ 术语顺序变换化。

$$\Phi_{LI}=\Phi_{L1}-\Phi_{L2}$$
$$R_{PI}=R_{P2}-R_{P1}$$

宽巷组合(Wide-laning combinations)

These combinations are used to create a signal with a significantly wide wavelength. This longer wavelength is useful for cycle-slips detection and ambiguity fixing. Other feature of this combination is the change of the sign in the ionospheric term, which is exploited by the Melbourne-Wübbena combination to remove the ionospheric refraction.

$$\Phi_{LW}=\frac{f_1\Phi_{L1}-f_2\Phi_{L2}}{f_1-f_2}$$

$$R_{PW}=\frac{f_1R_{P1}-f_2R_{P2}}{f_1-f_2}$$

窄巷组合(Narrow-laning combinations)

These combinations create signals with a narrow wavelength. The signal in this combination has a lower noise than each separated component. It is used to reduce the code noise in the Melbourne-Wübbena combination to estimate the wide-lane ambiguity.

$$\Phi_{LN}=\frac{f_1\Phi_{L1}+f_2\Phi_{L2}}{f_1+f_2}$$

$$R_{PN}=\frac{f_1R_{P1}+f_2R_{P2}}{f_1+f_2}$$

参考：https://gssc.esa.int/navipedia/index.php/Combination_of_GNSS_Measurements
