查看目录内容
  1. 终端实用技巧
  • 自动补全：
    在敲出”文件/目录 /命令前的几个字母后，按下tab键。如果输入的没有歧义，系统会自动补全，如果还存在其他 “文件/目录/命令 “，再按一下tab键(两次)，系统会提示可能存在的命令。
  • 使用过的命令：
    按”上/下”光标可以在曾经使用过的命令之间来回切换
    如果想退出，按ctrl+c
  • .代表当前目录
  • ..代表上一级目录
  1. ls命令说明

Linux下文件和目录特点

  • Linux文件或者目录名最长可以有 256个字符
  • 以. 开头 的文件为隐藏文件，需要用-a 才能显示
  • .代表当前目录
  • ..代表上一级目录(cd .. 可能到上一级目录)
  1. ls常用选项

-a 显示指定目录下所有子文件夹
-l 以列表的方式显示文件的详细信息
-h 配合-l以人性化显示文件大小

选项可以在一起用(例如ls -lha)

  1. 通配符的使用

星号 代表任意个数个字符
ls 1.txt 表示1开头的文件
ls 13.txt 表示以1开头3结尾的文件
ls *. 表示所有文件
? 代表任意一个字符，至少一个
ls ?2?.txt 表示中间是2的字符
[] 表示可以匹配字符组中的任一一个

[abc] 匹配a,b,c中的任意一个
ls [1234]23.txt
[a-f] 匹配从a到f范围内任意的一个字符
ls [1-4]23.txt

切换目录
  1. cd ,改变当前工作目录，也是用户最常用的命令之一。linux中所有的目录和文件大小写是敏感的

cd 切换当前用户的主目录(/home/用户名)
cd ~ 切换到当前用户的主目录(/home/用户目录)
cd . 保持当前用户的主目录
cd .. 切换到上级目录
cd- 可以在最近两个工作目录之间来回切换

  1. 相对路径和绝对路径
  • 绝对路径 输入绝对路径，最前面是/或者~，表示相对当前目录所在目录位置 cd /home/xdx/Desktop
  • 相对路径 最前面不是/或者~，表示当前目录所在的目录位置 cd Desktop/
  1. 创建和删除操作
  • 创建文件或修改文件时间
    如果文件不存在，可以创建一个空白文件
    如果文件存在，可以修改 文件的末次修改日期
  • mkdir
    创建一个新的目录
    -p可以递归创建目录(mkdir -p a1/b1/c1/d1)
    注意，新疆目录的名称不能与当前目录中已有的目录或文件同名
  • rm
    删除文件或目录(使用rm要小心，文件不能恢复)
    -f 强制删除，忽略不存在的文件，无需提示
    -r 递归删除目录下的内容，删除文件夹时必须加此参数
    rm 也可以加通配符(rm *1.txt，删除以1结尾的txt)

  1. tree

  2. cp命令
    cp命令的功能是讲给出的文件或者目录复制到另一个文件或者目录中。

    -f 已经存在的目标文件直接覆盖，不会提示
    -i 覆盖文件前提示
    -r 若给出的源文件是目录文件，则cp
    将递归复制目录下的所有子目录和文件，目标文件必须为一个目录名
